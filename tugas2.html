Belajar Bootstrap

Apa itu Bootstrap? Bootstrap adalah framework CSS open-source yang digunakan untuk membuat website responsif dengan cepat. Kenapa dipakai? Karena sudah menyediakan komponen siap pakai seperti grid system, tombol, navbar, dan layout yang rapi tanpa harus menulis CSS dari nol.
